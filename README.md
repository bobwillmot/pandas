# Pandas DataFrame Project

[![Python CI](https://github.com/bobwillmot/pandas/actions/workflows/ci.yml/badge.svg)](https://github.com/bobwillmot/pandas/actions/workflows/ci.yml)

A minimal Python project for working with pandas DataFrames using a local `venv` virtual environment.

## Requirements

- Python 3.14+

## Quick start

1. Create and activate the virtual environment:

```bash
python3 -m venv .venv
source .venv/bin/activate
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the sample script:

```bash
python src/main.py
```
